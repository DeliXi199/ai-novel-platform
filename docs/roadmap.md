## 当前默认工作流

1. 创建小说
2. 生成 global outline
3. 生成第一个 active arc
4. 用户手动触发下一章或连续生成 N 章
5. 每章生成后保存 summary，作为后续轻量上下文
6. 当前 arc 写完后，再按需生成下一段 arc
